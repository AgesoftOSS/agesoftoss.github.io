What is a Zero Day Vulnerability?

A Zero Day Vulnerability is a fresh discovered exploit in the software for which there is no patch because it's a newly discovered and the developers never heard about it before. "Day 0 after discovering"

How do they get discovered?

No System is safe. usually it's the hackers who are discovering these exploit and they immediantly exploit it. they get listed as a CVE (Common Vulnerabilities and Exposures)
