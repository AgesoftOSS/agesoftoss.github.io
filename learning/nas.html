What is a NAS?

NAS („Network Attached Storage“) is a storage device with a high storage capacity which uses the („Redundant Array Independent Discs“) RAID Technology to storage data. As the name says, it‘s connected to the internet and can only be used by authorized users. The NAS also can be accessed from a other location.

How does it work

It can run any Operating System (Windows, MacOS, Linux) and can be connected via RJ45 („Registered Jack“) Cable to the Network or through the WLAN („Wireless Local Area Network“)

Why do Companies use a NAS

The NAS is the first option for every company. The files on the NAS can be accessed all the time. It’s cheaper. It’s faster to save and load (depends on your network speed), And you’re independent from any third party solutions
