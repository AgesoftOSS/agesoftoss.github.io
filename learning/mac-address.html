Mac Address

Licencing

A Mac Address (Media Access Control Address) is a unique identifier assigned to the network interface controller (NIC) or commonly called "Networkcard" for identifying the NIC inside a network segment without a IP Address for example assigning a IP Address to a device without a identity with DHCP. A MAC Address is written in hexadecimal with 6 groups like this: "DB-D4-92-73-5B-83"
