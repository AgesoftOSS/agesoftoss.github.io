What is a Virtual Machine?

Virtual Machines (VM) is a isolated mashine inside of the host mashine that provides an isolated environment with it's guest operating system (OS). Virtual Machines are used to test things that may be risky on the host mashine

How does it work

Virtual Machines are just like the Host mashine except it's ran inside a virtualisation super visor software and isolated meaning it cannot access the data outside of the Virtual Mashine but you can enable it by configurating a Shared Folder.
