What is a DDOS?

A DDOS ("Distributed Denial of Service") is a malicious attack to disrupt the normal functioning of a computer network, service, or website by overwhelming it with a flood of traffic from multiple devices. It is used to take down websites & services that relay on unprotected or slow server hardware

How does it work?

Normally to access a internet website the visitor sends a request to the server. now imagine it when The server gets millions of requests per second. It overwhealms the server which takes the service down
