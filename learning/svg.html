What is a SVG?

SVG („Scalable Vector Graphics“) or simply Vector Graphics is a image file format that displays a image without quallity loss. As opposed to pixel-based graphics. Vector files store graphics via mathematical formulas. It can be resized and zoomed without losing any quallity and it stays sharp Vector graphic files are written in XML. Which helps site adminstrators rank their websites higher on search engines

Why should you use SVGs

Most common use for Vector Graphics are Company Logos. It also is mostly used in Illustrations and Artworks. Disadvantages of using SVG is they don’t get supported by old school browsers (IE8) and you require a extra software to edit or create these graphics most common softwares for SVGs are:

- Adobe Illustrator (PAID)
- Inkscape (FREE)
